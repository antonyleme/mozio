# Mozio Route Calculator

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Available Scripts

In the project directory, you can run:

To start the application:
### `yarn start`

To run the unit tests:
### `yarn test`

To run e2e tests:
### `yarn cypress:run`

To open cypress:
### `yarn cypress:open`
